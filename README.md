# FairFundz (ReactJS)

Pure ReactJS app (no Next.js), built with Vite + React Router, all components/pages in .jsx.

## Quickstart
1) cd fairfundz-react
2) npm install
3) npm run dev
4) Open http://localhost:5173

## Scripts
- npm run dev — start dev server
- npm run build — production build
- npm run preview — preview built app

## Tech
- React 18, React Router 6, Vite 5
- Minimal CSS with accessible, mobile-first layout
- 5 colors total: emerald (brand), white, slate-900, slate-600, amber (accent)

## Added UI stack
- Material UI (ThemeProvider, Buttons, Container)
- TailwindCSS utilities alongside minimal custom CSS
